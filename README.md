# WPF-Regex
正则提取
说明
=======
    
1.增加匹配列的显示

2.修改简单语法

    1. {} 直接输出 默认为 0， 可以 int   可以用 . 连接使用第二级的内容
    2. for 语句
    {for 开始标志 
        for 无参数时，表示循环输出全部
        for(n) 一个参数时，表示从0 开始 共输出 n 个
        for(m,) 第二个参数缺省时，表示从m 开始，输出后面所有的
        for(m,n) 两个参数时，表示从 m 开始，共输出 n 个
    循环体 {} 只能是 int 或 标签 ，不再支持 . 连接，暂不支持 for 循环 
    end} 结束标志
